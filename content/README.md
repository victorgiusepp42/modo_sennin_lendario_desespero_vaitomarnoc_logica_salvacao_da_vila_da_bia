# Conteúdo (formato intermediário IDL)

Arquivos fonte do material didático. Extensão sugerida: **`.idl.yaml`**.

## Regras

- Um arquivo pode representar um módulo, capítulo ou unidade — alinhar com o schema.
- Texto didático em blocos tipados (`paragraph`, `heading`, `list`, `callout`, etc.).
- Mídia referenciada por caminho relativo (ex.: `assets/figura-01.png`).

## Validar

```bash
npm run validate
```

Ver `example/modulo-exemplo.idl.yaml`.
